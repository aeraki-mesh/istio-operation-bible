腾讯云服务网格 TCM

TCM（Tencent Cloud Mesh）是腾讯云上提供的基于Istio进行增强，和Istio API完全兼容的Service Mesh托管服务，可以帮助用户以较小的迁移成本和维护代价快速利用到Service Mesh提供的流量管理和服务治理能力。
